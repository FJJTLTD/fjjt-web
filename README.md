# FJJT - Life Insurance Landing Page

A modern, responsive landing page for FJJT life insurance services.

## Features

- **Responsive Design** — Works on desktop, tablet, and mobile devices
- **Modern UI** — Clean green color scheme with professional typography
- **Quote Form** — Collects customer information for insurance quotes
- **Feature Highlights** — Showcases key benefits (Affordable Coverage, Quick & Easy, Trusted Protection)
- **Benefits Section** — Explains how life insurance can help (Replace Income, Pay off Debt, Future Goals, Final Expenses)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize content, colors, and images as needed

## Deployment

### GitHub Pages
1. Go to your repository Settings
2. Scroll to "GitHub Pages"
3. Select `main` branch as source
4. Your site will be live at `https://FJJTLTD.github.io/fjjt-web`

## Customization

- **Colors** — Edit the hex values in the `<style>` section
- **Text** — Update headings and descriptions directly in the HTML
- **Images** — Add an image URL to the `<img>` tag in the hero section
